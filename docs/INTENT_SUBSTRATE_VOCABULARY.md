# Intent Substrate Vocabulary

This is the canonical lexicon for the Drovi new paradigm. It is enforced in docs, UI copy, and technical architecture.

| Old Term | New Term | Definition |
| --- | --- | --- |
| Missions | Continuums | Persistent goal processes that run until completion or cancellation. |
| Syscalls | Actuators | Capability primitives that execute actions across tools. |
| Knowledge Graph | Reality Fabric | Bi‑temporal, evidence‑anchored world state. |
| Verification Engine | Proof Core | Evidence‑anchored verification mechanism. |
| Context Allocator | Context Loom | Attention budgeter and retrieval kernel. |
| Mission Control | Command Deck | UI for running Continuums and escalations. |
| Timeline | Reality Stream | Time‑indexed playback of truth with proof. |
| Omnibox | Intent Bar | Universal command surface. |
| Actions | Executions | Policy‑gated, reversible operations. |
| Evidence | Proof | Immutable source references tied to claims. |
