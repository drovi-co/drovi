# AgentOS Pilot Enablement Package

## Objective
Enable pilot organizations to deploy Sales/HR/Legal/Accounting AI employees with governance and measurable outcomes in week 1.

## Package Contents

## 1) Onboarding Guide
1. Create org and admin users.
2. Connect sources (email, calendar, docs, CRM, chat).
3. Install starter packs from Agent Catalog.
4. Configure approval chains and denylist policy.
5. Dispatch first supervised runs and validate evidence links.

## 2) Governance Templates
- Default autonomy tiers by function:
  - Sales/HR: `L2` (low-risk execute).
  - Legal/Accounting: `L1-L2` with strict approvals.
- Approval policy baseline:
  - external send, contract edits, and financial writes require approval.
- Emergency controls:
  - denylist switch + deployment rollback.

## 3) Role Pack Recommendations
- Sales:
  - SDR Agent
  - RevOps Hygiene Agent
  - Renewal Risk Agent
- HR:
  - Recruiting Coordinator
  - Onboarding Manager
  - Policy Drift Agent
- Legal/Accounting:
  - Advice Timeline Sentinel
  - Contradiction Detection Agent
  - Filing & Missing Docs Agent

## 4) Success Metrics (Pilot Week 1-4)
- run success rate
- approval SLA
- evidence coverage rate
- quality drift score
- concrete work products delivered

## 5) Weekly Pilot Cadence
1. Monday: review failures + backlog + drift.
2. Wednesday: policy/prompt tuning.
3. Friday: ROI review (time saved, work products delivered, risk reduced).

## 6) Exit Criteria
- At least 3 production-critical workflows automated with approval controls.
- SLO targets stable for 2 consecutive weeks.
- Pilot stakeholders approve expansion plan.
